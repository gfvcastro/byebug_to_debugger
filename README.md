# byebug_to_debugger

A `byebug` to `debugger` alias.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'byebug_to_debugger', group: [:development, :test]
```

And then execute:

```bash
bundle install
```

Or install it yourself as:

```bash
gem install byebug_to_debugger
```
